description: Aturan commit dan git — dipakai setiap kali agent akan melakukan commit
---

- Commit setelah tahap selesai DAN sudah direview user — bukan otomatis di tengah tahap
- Format pesan commit: `tahap N: <ringkasan singkat>`
- Jangan pernah force-push atau reset history tanpa diminta eksplisit
