---
name: boundaries
description: Batasan keras yang tidak boleh dilanggar agent dalam kondisi apapun
---

- Jangan mengerjakan lebih dari satu tahap dalam satu response kecuali diminta eksplisit
- Jangan mengubah migration yang sudah di-migrate — buat migration baru untuk perubahan
- Jangan menghapus/menimpa file di luar scope tugas yang diberikan
- Jangan menambah package/library baru tanpa menyebutkan alasan ke user dulu
- Jika requirement ambigu, tanya user dulu — jangan berasumsi
